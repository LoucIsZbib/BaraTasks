Additional Credits

Nitro uses a number of additional libraries and resources.

  • Written in CoffeeScript and SASS
  • Structured using the Base MV* framework
  • Drag and drop magic using Mouse
  • Compiling files together with Scrunch
  • Manipulating the DOM with jQuery
  • Sending realtime messages with Socket.IO
  • Icongraphy from GLYPHICONS
  • Picking dates with the jQuery UI datepicker
  • Expanding notes with jQuery Autosize
